## 二、已完成功能

### 1. 认证与权限
- 账号密码登录，JWT Token 验证
- 五种角色：超级管理员、医院管理员、科室管理员、医生、放射科医生
- 登录后按角色自动路由（门诊工作台 / 住院工作台 / 影像工作台 / 后台管理）
- 所有关键操作写入审计日志

---

### 2. 门诊接诊工作台

**接诊管理**
- 新建接诊（自动创建患者 + 接诊记录）
- 续接诊（恢复进行中的接诊）
- 历史病历查看

**问诊录入面板**
- 主诉（必填）、现病史、既往史、过敏史、个人史
- 月经史（女性患者自动显示）
- 生命体征快速录入：T / P / R / BP / SpO₂ / 身高 / 体重，一键填入体格检查
  - 多次填入智能合并（保留已有项，追加新项）
- 辅助检查：快速开单（34 项检验，分 6 类）+ 上传检验报告（AI OCR）
- 初步印象

**保存逻辑**
- 保存按钮脏状态检测（有修改 → 蓝色"保存问诊信息"，已保存 → 绿色"已保存 ✓"禁用）
- 保存后自动同步所有字段到已生成的病历对应段落
- 填入体格检查 / 快速开单不触发 AI 刷新

---

### 3. 住院接诊工作台

与门诊同等功能，额外包含：
- 婚姻史、家族史、知情人、目前用药
- 入院专项评估：VTE 风险、营养评估、心理评估、康复评估、宗教信仰、疼痛评估
- 入院诊断字段

---

### 4. AI 功能模块

| 功能 | 说明 |
|------|------|
| 语音录入 | Web Speech API 实时转写 + 音频保存 + AI 结构化分析（医患对话分离） |
| 追问建议 | 保存主诉后自动生成，带快捷回答选项，支持获取更多 |
| 检查建议 | 按基础必查 / 鉴别诊断 / 高风险分类展示 |
| 诊断建议 | AI 生成可能诊断（高度符合 / 可能符合 / 待排除），一键写入初步印象 |
| 一键生成病历 | 门诊 / 住院两种模板，流式输出 |
| 续写 / 润色 | 追加内容或润色已有病历，均流式输出 |
| AI 质控扫描 | 检查完整性、格式、逻辑、规范性、医保风险五类问题 |
| 质控修复 | AI 生成修复建议，支持手动编辑后一键写入病历 |
| 甲级评分 | 0-100 分环形进度图，显示甲级 / 乙级 / 丙级 |

---

### 5. 病历编辑器

- 中间区域富文本编辑（Ant Design TextArea）
- 一键生成 → 续写 → 润色 → AI 质控 → 出具最终病历 完整流程
- 病历类型切换（门诊病历 / 住院病历）
- 版本历史记录（每次保存生成新版本）
- 出具最终病历后锁定，不可再修改

---

### 6. 影像分析（PACS / DICOM）

- 上传 DICOM ZIP / RAR 压缩包，自动解压 + 缩略图生成
- DICOM 查看器（Canvas 渲染，支持窗位 / 窗宽调节）
- 智能 18 帧抽帧算法（头尾各取 3 帧，中部密集取 12 帧，优化脊柱 CT 覆盖）
- AI 分析（千问 VL 模型）+ 报告编辑 + 一键发布
- 临床医生可直接上传 JPG / PNG 快速分析，结果插入辅助检查

---

### 7. 检验报告上传（本期新增，待真实测试）

- 辅助检查旁「上传报告」按钮，支持 JPG / PNG / PDF
- AI OCR 识别（千问 VL）：提取项目名、结果值、参考范围、异常标注
- 识别结果一键插入辅助检查字段
- 右侧面板「检验报告」Tab：查看本次接诊所有报告，支持再次插入 / 删除

---

### 8. 后台管理（13 个页面）

| 页面 | 功能 |
|------|------|

| 用户管理 | 增删改查医生 / 管理员账号 |

| 病历管理 | 搜索 / 查看所有病历 |

| 模型配置 | 按场景切换 AI 模型（生成 / 润色 / 质控 / 问诊 / 检查） |
| 数据统计 | 按科室接诊量、7 日趋势、质控问题分布 |
| Token 用量 | DeepSeek / 阿里云余额 + 本地 Token 消耗统计 |
| 语音记录 | 查看所有语音转写记录 |
| 审计日志 | 所有操作日志 |
